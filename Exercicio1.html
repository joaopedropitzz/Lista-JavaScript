O que é JavaScript?

JS é uma linguagem de programação interptetada, para desenvolvimento web que é executada no navegador do client em um sandbox, que limita o acesso aos recursos do sistema operacional, fornecendo interfaces predefinidas. A motivação por traz do Javascript foi tornar as páginas web mais dinâmicas e interativas, sendo que na época o único meio de interação era por meio de links e formulários.
Diferença entre execução no navegador e no servidor: A principal diferença está em que lado o programa é executado, enquanto no navegador o programa é executado no dispositivo do usuário, no servidor o código é rodado em uma máquina remota.
Sandbox: sandbox é um ambiente seguro e isolado para execução de código, tendo como seu objetivo principal a segurança do usuário.
Elementos globais, document, console e alert: objetos globais são responsáveis por interagir com a página web. O document representa a estrutura da página, o console fornece métodos e dados no console do navegador e o alert exibe uma caixa de diálogo com uma mensagem
